# TagItemSeller

TagItemSeller is an addon for World of Warcraft that allows players to easily tag items in their inventory for automatic selling. With a simple Ctrl+right-click on an item, TagItemSeller marks it for sale. When you interact with a vendor in the game, all tagged items are sold automatically, streamlining inventory management and helping you earn gold faster.
